同じブラウザで画面を見るとき、**フル版のツールバーは相手にも見える**。相手には **甲記入用** を渡す。

1. **乙の連絡先** … **HTML 内の `PROVIDER_DEFAULTS` に固定**（`provider-contact.md` の表と揃える）。変更するときは HTML を編集し直す。ブラウザの自動保存の対象外（甲の入力だけが保存される）。
2. **同じPC・同じブラウザ** … 入力は自動でブラウザに保存される（閉じても残ることが多い）。
3. **別のPCでも乙の内容を使いたい** … **「入力内容をファイルに保存」** で `契約書_AI業務再設計30日伴走-乙マイ設定.json` をダウンロードし、クラウド等で持ち運び、もう一方のPCで **「ファイルから読み込み」** で復元する。
4. **相手（甲）** … **`fillable-甲記入用.html`** を渡す（ネット不要）。JSONは乙用なので甲には渡さない。

※ JSON には個人情報が含まるため、公開リポジトリにコミットしないこと。
